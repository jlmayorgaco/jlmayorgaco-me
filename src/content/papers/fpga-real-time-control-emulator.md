abstract: "This paper presents the development of a real-time control emulator using FPGA technology and the HiLeS (Hardware-in-the-Loop Simulation) methodology. The system implements control algorithms in hardware for deterministic execution, bypassing the non-real-time limitations of software-based controllers. The emulator supports multiple control strategies including PID, state feedback, and adaptive control, with configurable plant models."
