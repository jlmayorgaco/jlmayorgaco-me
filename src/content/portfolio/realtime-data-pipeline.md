# Real-time Data Pipeline

A scalable real-time data processing pipeline for industrial IoT applications.

## Overview

End-to-end data pipeline from edge devices to cloud storage with real-time analytics and alerting.

## Technical Details

- **Edge**: Rust-based processing on embedded Linux
- **Transport**: MQTT, WebSocket
- **Stream**: Apache Kafka
- **Storage**: InfluxDB + PostgreSQL

## Features

- 100k+ messages/second throughput
- Local processing for low latency
- Cloud sync with conflict resolution
- Configurable retention policies
