# CTS - Traffic Control System

A distributed traffic control system designed to optimize traffic flow in urban environments using real-time control algorithms.

## Overview

This project implements a smart traffic light system that uses sensors to detect vehicle density and adjusts timing dynamically to reduce congestion.

## Technical Details

- **Microcontroller**: STM32F4 with FreeRTOS
- **Sensors**: Ultrasonic distance sensors, inductive loop detectors
- **Communication**: CAN bus for inter-node communication
- **Control Algorithm**: Fuzzy logic based adaptive control

## Results

- 30% reduction in average wait times
- Real-time adaptability to traffic patterns
- Scalable architecture for city-wide deployment
