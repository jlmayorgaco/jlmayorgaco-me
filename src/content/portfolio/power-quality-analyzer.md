# Power Quality Analyzer

A real-time power quality monitoring system for industrial and commercial installations.

## Overview

Continuous monitoring of power quality parameters including voltage, current, harmonics, flicker, and transients.

## Technical Details

- **ADC**: 16-bit, 128 kS/s
- **Processing**: FFT-based harmonic analysis
- **Standards**: IEC 61000-4-30 Class A
- **Display**: Real-time waveform and trends

## Features

- 50th harmonic analysis
- Event capture (sags, swells, interruptions)
- Flicker measurement (Pst, Plt)
- Configurable alarms and thresholds
