# 🚀 jlmayorgaco-me | Personal Research & Engineering Hub

Welcome to my personal brand engine. This repository is more than just a website; it's a **self-curating R&D pipeline** designed to bridge the gap between Senior Web Engineering and Advanced Control Systems Research.

## 🤖 The Architecture
This project automates the "Literature Review to Publication" loop:
- **Fetch:** Automated daily scans of ArXiv & IEEE for MAS, FPGA, and Power Systems papers.
- **Synthesize:** AI-powered summarization (Gemini 1.5 Flash) for high-signal technical insights.
- **Curate:** Telegram-integrated feedback loop for human-in-the-loop expert commentary.
- **Publish:** Zero-friction deployment to my Astro-based site via GitHub Actions.

## 🛠 Tech Stack
- **Frontend:** Astro (SSG) + Tailwind CSS + TypeScript.
- **Automation:** Python 3.12 + GitHub Actions (CRON).
- **Intelligence:** Google Gemini API (LLM).
- **Hosting:** Cloudflare Pages (Edge Delivery).
